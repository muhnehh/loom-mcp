Get started in 30 seconds.

One command. Then configure your AI.

1

Install

Globally via npm, or use npx.

npm install -g @loom-mcp/server
2

Initialize

Scan your codebase and see the savings.

loom init
3

Configure your AI

Add the MCP server to your preferred tool.

# Claude Code /mcp add loom localhost:8080 # Cursor - Add to settings.json { "mcpServers": { "loom": { "command": "loom", "args": ["start"] } } }
4

Done.

Open http://localhost:2337 to watch your savings.

Full MCP Toolkit (28+)

Click any tool name to expand and view what it does.

  • loom_get_topology

    Returns an AST skeleton of the codebase with signatures and structure, without full implementations.

  • loom_focus

    Pages in the full implementation for a specific file or symbol so the agent can work with exact code.

  • loom_search_symbols

    Searches for symbols by name and relevance to quickly locate functions, classes, and methods.

  • loom_get_symbol

    Retrieves the exact source for a symbol using fast byte-offset lookup.

  • loom_find_importers

    Finds files that import or depend on a symbol to map reverse dependencies.

  • loom_blast_radius

    Estimates change impact by showing what parts of the code are affected by editing a symbol.

  • loom_search_refs

    Performs AST-aware reference search to find real code usages, not just text matches.

  • loom_bm25_search

    BM25 ranking - statistically optimal keyword search better than simple match.

  • loom_fuzzy_search

    Fuzzy/Levenshtein matching for typos and partial matches.

  • loom_find_dead_code

    Detect unused functions and unreachable code.

  • loom_pagerank_centrality

    PageRank algorithm for architectural importance scoring.

  • loom_get_class_hierarchy

    Traverse class inheritance tree - get parents and children.

  • loom_gpu_search

    Real GPU semantic search with CUDA via @xenova/transformers.

  • loom_watch_start/stop

    Live file watching with auto-reindex on changes.

  • loom_attach_repo

    Attach multiple repos for cross-repo search.

  • loom_semantic_search

    GPU-accelerated semantic embeddings (optional).

  • loom_detect_frameworks

    Detect Nuxt/Next.js/React/Django/etc.

  • loom_enforce_hook

    Register PreToolUse/PostToolUse hooks to force agent tool usage.

  • loom_remember

    Saves a useful code insight as cross-session memory for future runs.

  • loom_recall

    Retrieves stored cross-session insights based on a query.

  • loom_get_deps

    Builds dependency graphs in text, DOT, or JSON formats to visualize system relationships.

  • loom_gateway_notify

    Send Slack/Discord notifications.

  • loom_workspace_stats

    SQLite workspace statistics.

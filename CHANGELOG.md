# Changelog

All notable changes to LoomMCP will be documented in this file.

## [0.6.0] - 2026-04-25

### Added
- **GPU Semantic Search** - Real CUDA embeddings via `@xenova/transformers`
- **Live Watch** - Auto-reindex on file changes with debouncing
- **SQLite Workspace** - Single-file DB with FTS5 (like Srclight)
- **LoomPacks** - Pre-configured framework toolkits (React, Node, Django, Rust, Go, Java, C#)

### Tools Added
- `loom_gpu_search` - GPU semantic embeddings
- `loom_gpu_status` - GPU availability check
- `loom_watch_start/stop` - Live file watching
- `loom_workspace_stats/clear` - SQLite workspace
- `loom_bm25_search` - BM25 ranking search
- `loom_fuzzy_search` - Fuzzy/Levenshtein matching
- `loom_find_dead_code` - Unused code detection
- `loom_pagerank_centrality` - Architectural importance
- `loom_get_class_hierarchy` - Class inheritance tree
- `loom_attach_repo` - Multi-repo attachment
- `loom_semantic_search` - GPU-accelerated embeddings
- `loom_detect_frameworks` - Nuxt/Next.js/Django detection
- `loom_get_confidence` - Methodology disclosure
- `loom_enforce_hook` - PreToolUse/PostToolUse hooks

### Languages
- TypeScript/JavaScript, Python, Rust, Go, Java, C#, C/C++, Ruby
